# Synthetic Test Data

Generated records for specific test scenarios. Unlike downloaded data, synthetic records are committed to git because they are small, version-controlled, and reproducible.

Regenerate with `just generate-synthetic`.

## Directories

- `malformed/` — Intentionally broken records for negative testing
- `encoding/` — Encoding test vectors for roundtrip verification
- `generators/` — Python scripts that produce the .mrc files

## malformed/

| File | Contents | Expected parse behavior |
|------|----------|------------------------|
| `truncated_leader.mrc` | Records with leaders < 24 bytes | Err or Ok(None) |
| `invalid_lengths.mrc` | Leader length doesn't match actual size | Err (truncated/length mismatch) |
| `bad_directory.mrc` | Corrupted dir entries (non-digits, incomplete, overlapping) | Err (directory parse failure) |
| `missing_terminators.mrc` | Records missing field/record terminators | Err or partial parse |
| `embedded_terminators.mrc` | 0x1D/0x1E in unexpected positions | Err or truncated/split parse |
| `garbage.mrc` | All-zeros, all-0xFF, pseudo-random bytes | Err or Ok(None) |

Generated by `generators/generate_malformed.py` using raw binary construction (no mrrc API).

## encoding/

| File | Contents | Expected parse behavior |
|------|----------|------------------------|
| `utf8_titles.mrc` | UTF-8 titles: ASCII, diacritics, CJK, Cyrillic, umlauts | Clean roundtrip, no U+FFFD |
| `marc8_ascii.mrc` | Leader pos 9 = ' ' (MARC-8), pure ASCII content | Clean roundtrip, leader preserved |
| `replacement_chars.mrc` | Pre-existing U+FFFD in title fields | U+FFFD survives roundtrip |
| `mixed_scripts.mrc` | Multiple scripts in single title (e.g. "English 中文 Москва café") | Clean roundtrip |

Generated by `generators/generate_encoding.py` using mrrc's Python API (Record, Field, MARCWriter).
